# HTTP STATUS CODE
클라이언트가 HTTP 리퀘스트를 보낸 결과, 즉 서버가 정상적으로 처리되었는지, 에러가 발생했는지 알려주는 것이 HTTP상태코드 입니다.

1XX : 조건부 응답

2XX : 성공

3XX : 리다이렉트 

4XX : 요청 오류

5XX : 서버오류
